# Notebooks

This directory contains Jupyter notebooks for experiments, analysis, and demonstrations.

## Notebooks

### Getting Started
- `01_environment_demo.ipynb`: Demonstration of the trading environment
- `02_black_scholes_basics.ipynb`: Black-Scholes model and delta hedging

### Training and Experiments
- `03_train_q_learning.ipynb`: Training Q-learning agent
- `04_train_dqn.ipynb`: Training Deep Q-Network agent
- `05_hyperparameter_tuning.ipynb`: Hyperparameter optimization

### Analysis
- `06_agent_comparison.ipynb`: Comparing different RL agents
- `07_transaction_cost_impact.ipynb`: Impact of transaction costs on hedging
- `08_results_visualization.ipynb`: Visualizing experiment results

## Usage

Start Jupyter:
```bash
jupyter notebook
```

Navigate to this directory and open any notebook.

## Best Practices

1. **Clear outputs before committing**: Cell outputs can make notebooks large
2. **Document your experiments**: Add markdown cells explaining your approach
3. **Use descriptive names**: Name notebooks clearly
4. **Save results separately**: Don't store large results in notebooks
